# Changelog
## [Unreleased]

## [0.1.0] - 2026-04-03
### Added
- Added configurable writing profiles and custom vocabulary settings.
- Added custom transition words, sequence markers, and auto-format paragraph break rules.

### Changed
- Auto Format now follows profile-aware and user-defined paragraph breaks.
- Updated metadata and documentation to position Powered Plain Text as a writing aid for structured text and LaTeX.
- Logic-word styling is now handled at runtime, while the TextMate grammar focuses on stable text patterns.

### Fixed
- Reduced overlapping and false-positive logic or sequence hints.
- Kept settings and visible styling aligned.

## [0.0.2] - 2025-12-13
### Added 
- Compatibility with latex files
- logo

### Changed
- Renamed project name to "Powered Plain Text"

## [0.0.1] - 2025-11-28
### Added
- Initial release of the "Powered Plain Text" extension with automatic syntax highlighting and logical symbol hints for plain text documents. Supported file extensions include `.txt`, `.rtxt`, and `.rich`.
- Added features for recognizing logical transition words, punctuation, quoted content, book titles, parenthetical content, and numbers with distinct highlighting styles.
- Implemented logical symbol hints for various transition words to enhance readability.
